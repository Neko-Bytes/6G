# 3GPP Specification Reference

This document serves as a reference for the 3GPP specifications related to the 5G Core (5GC) architecture.

| Specification Number | Heading | Content Summary | Relation to 5G Core Architecture |
| :--- | :--- | :--- | :--- |
| **TS 23.501** | System architecture for the 5G System (5GS); Stage 2 | Defines the overall 5G system architecture. It specifies Network Functions (AMF, SMF, UPF, PCF, UDM, etc.), reference models (non-roaming, roaming, data storage, service-based interfaces), and high-level features such as network slicing and edge computing. | Foundational document defining the structure, components, and interfaces of the 5G Core. |
| **TS 23.502** | Procedures for the 5G System (5GS); Stage 2 | Provides detailed procedures and message flows for 5GS operations. Covers registration management, connection management, mobility management, and session management (PDU session establishment, modification, and release). | Specifies the dynamic interactions and signaling flows between network functions to support 5G services. |
| **TS 29.503** | Unified Data Management Services; Stage 3 | Defines the protocol-level implementation of UDM services. It specifies Service Based Interfaces (SBIs) such as Subscriber Data Management (SDM), UE Context Management (UECM), and UE Authentication (UEAU), including their OpenAPI definitions. | Provides the technical details for how the UDM manages subscriber data and authentication within the 5G Core. |
